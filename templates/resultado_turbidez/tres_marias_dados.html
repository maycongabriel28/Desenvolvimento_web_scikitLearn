Análise reservatório de Três Marias/MG

Para fazer a predição de parâmetros limnólogicos no reservatório de Três Marias, usou dados históricos de coletas de amostras de água, dados espectrais de imagens do satelite Landsat 8 OLI, algoritmos de machine learnig. Foi feito o treinamento de algoritmos de machine learning para fazer a predição do parâmetro turbidez.

Para fazer o treinamento do algoritmo de machine learning regressão linear, usou informações espectrais das bandas 2, 3, 4 do Landsat 8 OLI e também os dados históricos de parâmetros limnólogicos cor verdadeira, sulfato e turbidez (Zona fótica).

Predição de turbidez-mg/l

Escolha um dentre os parâmetros abaixo para fazer a predição de turbidez:

  • B2 - Banda 2
  • B3 - Banda 3
  • B4 - Banda 4
  • cor_verdadeira_S - Superfície
  • sulfato_S - Superfície
  • turbidez_ZF - Zona fótica

OBS: Preencha o campo "Informe parâmetro para predizer turbidez" com os parâmetros iguais aos da lista acima.
